KROK 1 — SUPABASE (baza danych)
1. Wejdź na supabase.com → Start for free
2. Zaloguj się → New Project → czekaj ~1 min
3. Settings → API → skopiuj Project URL i anon public key
KROK 2 — SQL EDITOR w Supabase
Wklej i uruchom (przycisk Run / F5): CREATE TABLE workouts ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, user_id text NOT NULL, training_id text, exercise_id text, series jsonb, created_at timestamptz DEFAULT now() ); ALTER TABLE workouts ENABLE ROW LEVEL SECURITY; CREATE POLICY "public" ON workouts FOR ALL USING (true);
