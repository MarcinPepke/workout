WORKOUT TRACKER
Jednorazowa konfiguracja — zajmie ~3 minuty
KROK 1 — SUPABASE (baza danych)

1. Wejdź na supabase.com → Start for free
2. Zaloguj się przez GitHub lub email
3. Kliknij New Project, podaj nazwę np. "workout"
4. Poczekaj ~1 min na utworzenie
5. Idź do Settings → API
6. Skopiuj Project URL i anon public key

KROK 2 — UTWÓRZ TABELĘ

W Supabase idź do SQL Editor i wklej oraz uruchom ten kod:

CREATE TABLE workouts (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id text NOT NULL,
  training_id text,
  exercise_id text,
  series jsonb,
  created_at timestamptz DEFAULT now()
);
ALTER TABLE workouts ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public" ON workouts FOR ALL USING (true);
